## Summary
- Introduced a full external-SSD setup runbook for Docker Desktop users.
- Added `dockguard stop` with optional `--eject` to support safer shutdown workflows.
- Improved storage-path compatibility by accepting case-insensitive Docker Desktop settings keys.

## Highlights
- New runbook: `docs/docker-local-setup.md`
- New command path: `dockguard stop [--eject]`
- Better compatibility with real Docker Desktop settings layouts (for example `DataFolder`)

## Notes
- This release is focused on safe external storage operations and practical macOS workflows.
