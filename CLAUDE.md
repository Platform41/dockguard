# DockGuard - Local AI Notes

This repo can be used without any private tooling.

Optional local tools:
- If you have access to the 41 Studio foundation OS, you can add it at `.os/` locally.
- `.os/` is ignored by git and is not required to build or run DockGuard.

Basics:
- Go version: follow `go.mod`.
- Run the app: see `README.md`.

Conventions:
- Do not commit secrets or local config.
